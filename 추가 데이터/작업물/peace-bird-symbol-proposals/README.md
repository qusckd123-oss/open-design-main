# Peace Bird Symbol Proposals

Reference mood: minimal peace bird / swallow silhouette, generous negative space, poster-like monochrome graphics.

## Candidates

- A. Soft Dove: 가장 평화 상징성이 직관적인 둥근 실루엣입니다.
- B. Ribbon Wing: 안쪽 여백을 리본처럼 정리해 심볼성이 강합니다.
- C. Blue Orbit: 파란 원을 더해 그래픽 포인트와 캠페인 확장성이 있습니다.
- D. Negative Cut: 흰 여백이 날개 방향을 만들며 로고 적용에 좋습니다.
- E. Emblem Curve: 둥근 배지/와펜/프린트에 안정적인 구조입니다.
- F. Poster Grain: 포스터와 의류 프린트에 어울리는 질감형입니다.

## Files
