# Key Item Slice Library

Each item family naturally exposes a different subset of the 5 slices.
Pick the slices that the *item* answers, not the slices that look pretty.
Generic 5-slice spreads are a tell of AI-slop — real studio sheets pick
2–4.

## The 5 slices

| Slice | What it shows | When to include |
|---|---|---|
| **Color** | Same garment, multiple colorways, hanger or flat-lay | Any item with multi-color development; mandatory for knits, denim setups, basics |
| **Detail** | Macro close-up of one construction element (cuff, hem, pocket, drawcord, stitching, zipper pull) | Anything carrying a signature detail; mandatory for new-basics, easy-denim setup, reissued silhouettes |
| **Pattern** | Pattern at swatch scale: woven (check, stripe, dot), printed (camo, floral), structural (waffle, ribbed) | Items where the pattern is the story (check shirts, polka-dot series, camo, ruffle / lace) |
| **Fabric** | Swatch close-up showing hand + drape, optional fabric-tag overlay | Items where the fabric is the upgrade (functional cooling, Sorona, denim weight, slub jersey) |
| **Artwork** | Graphic / embroidery / patch / sticker / wordmark study isolated on garment area | Items where the artwork is the value (signature graphic tee, vintage graphic tee, jelly logo lockup) |

## Recommended composition per item family

| Item family | Default slices | Why |
|---|---|---|
| Outer — cotton zip-up jumper | Washing · Pattern · Artwork | Outer story is *worn surface texture* — wash variation, plaid pattern, embroidered chest patch |
| Outer — windbreaker / nylon | Color · Detail · Fabric | Function is the value — color, drawcord/zip detail, lightweight fabric |
| Top — signature graphic tee | Artwork · Fabric | Value is graphic + cooling fabric (Cool Cotton, Askin); silhouette is generic |
| Top — vintage graphic tee | Graphic · Fabric (Pigment Dyeing as fabric note) | Faded color + slub jersey + retro graphic = the whole item |
| Top — basic / new-basic tee | Color · Detail · Fabric | Value is fit + neckline + jersey grade |
| Knit — cardigan, sweater, vest | Color · Pattern · Detail · Fabric | Knits split across all four naturally |
| Bottom — easy-denim setup | Color · Detail · Fabric | Color label, signature stitching/embroidery, functional denim (Coolmax, Linen-like) |
| Bottom — sweatpant / shorts | Color · Detail · Fabric | Same as above |
| Setup — denim-like sweat setup | Color · Detail · Fabric (Washing as a sub-note inside Color) | Setup needs colorway story + signature detail |
| Dress / skirt | Color · Detail · Pattern | Pattern is often the story (polka-dot, floral, lace) |
| Hat / cap | Color · Detail · Artwork | Cap value = color, brim/strap detail, embroidery |
| Bag / backpack | Color · Detail · Pattern (or Artwork) | Detail = strap, zipper, ring; pattern when applicable |
| Shoes (sneaker, sandal, slide) | Color · Detail · Fabric | Material upper + sole detail |

## Slices to *not* mix

- **Color × Pattern** when the patterns include color variation — they
  will fight visually. Show pattern in monochrome, then handle color in
  the Color slice.
- **Detail × Artwork** for tees — they read as the same thing at thumb
  scale. Pick one.
- **Pattern × Artwork** for outerwear with embroidered patches on
  patterned fabric — pick whichever the season hero is.

## Slice naming variants

Some studios use locale-specific slice names. Map them onto the canonical
five in the template:

| Studio name | Canonical slice |
|---|---|
| Washing | Color (treat as a Color sub-axis) |
| Wash | Color |
| Print | Pattern *or* Artwork (Pattern when repeating, Artwork when isolated) |
| Embroidery | Artwork |
| Trim | Detail |
| Hardware | Detail |
| Finish | Fabric *or* Color (depending on whether finish changes hand or color) |
| Material | Fabric |
| Knit structure | Pattern (when ribbed/cable visible) *or* Fabric (when about hand) |
