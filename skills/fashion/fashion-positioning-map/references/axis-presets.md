# Positioning Map Axis Presets

Six common 2x2 axis pairings from K-young-casual / streetwear strategy
work. Pick **one** in Step 1 or define a custom pair using the same
shape (left-label, right-label, top-label, bottom-label, 1-line
guidance for placement).

---

## P1. Industry standard — BASIC ↔ ACCENT × NEW BASIC ↔ TREND

- **x-axis** — BASIC (left) ↔ ACCENT (right)
- **y-axis** — TREND (top) ↔ NEW BASIC (bottom)
- **Use when** — K-young-casual / streetwear default; the WACKYWILLY
  27SS deck uses this
- **Placement guidance** —
  - Pure-basic uniform brands → bottom-left
  - Hype / accent-heavy streetwear → top-right
  - Quiet new-basic brands → bottom-center
  - Trend-leaning seasonal brands → top-center

---

## P2. Price × Trend — VALUE ↔ PREMIUM × CLASSIC ↔ TREND

- **x-axis** — VALUE (left) ↔ PREMIUM (right)
- **y-axis** — TREND (top) ↔ CLASSIC (bottom)
- **Use when** — multi-tier brand portfolio strategy; comparing
  value-tier vs. luxury arms
- **Placement guidance** —
  - Fast-fashion → top-left
  - Luxury hype → top-right
  - Value classics (Uniqlo) → bottom-left
  - Heritage premium → bottom-right

---

## P3. Mass × Identity — BROAD ↔ NARROW × COMMODITY ↔ SIGNATURE

- **x-axis** — BROAD audience (left) ↔ NARROW audience (right)
- **y-axis** — COMMODITY (top) ↔ SIGNATURE (bottom)
- **Use when** — assessing distinctiveness vs. reach
- **Placement guidance** —
  - Mass commodity (Hanes) → top-left
  - Mass signature (Nike) → bottom-left
  - Niche commodity (rare) → top-right
  - Niche signature (Comme des Garçons) → bottom-right

---

## P4. Era × Tone — RETRO ↔ CONTEMPORARY × QUIET ↔ LOUD

- **x-axis** — RETRO (left) ↔ CONTEMPORARY (right)
- **y-axis** — QUIET (top) ↔ LOUD (bottom)
- **Use when** — concept-board era × tone alignment
- **Placement guidance** —
  - Vintage workwear → top-left
  - Modern minimal → top-right
  - 90s revival streetwear → bottom-left
  - Hype contemporary → bottom-right

---

## P5. Gender × Form — MASCULINE ↔ FEMININE × STRUCTURED ↔ FLUID

- **x-axis** — MASCULINE-coded (left) ↔ FEMININE-coded (right)
- **y-axis** — STRUCTURED (top) ↔ FLUID (bottom)
- **Use when** — bridge / capsule planning between UNI and WOMEN
- **Placement guidance** —
  - Tailored menswear → top-left
  - Tailored womenswear → top-right
  - Sportswear menswear → bottom-left
  - Soft womenswear → bottom-right

---

## P6. Function × Mood — UTILITY ↔ EXPRESSION × COOL ↔ WARM

- **x-axis** — UTILITY-led (left) ↔ EXPRESSION-led (right)
- **y-axis** — COOL palette (top) ↔ WARM palette (bottom)
- **Use when** — palette × posture review for season planning
- **Placement guidance** — useful for aligning with concept-board
  territory

---

## Shape for a custom pair

```
Name              — short pair name for the map title
x-axis            — LEFT-LABEL ↔ RIGHT-LABEL
y-axis            — TOP-LABEL ↔ BOTTOM-LABEL
Use when          — when to pick this pair
Placement guide   — 3–6 anchor reference brands per quadrant
```
