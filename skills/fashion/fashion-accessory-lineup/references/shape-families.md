# Accessory Shape Family Library

Canonical shape families per accessory category. Pick **2–4** in
Step 1 or define a new family using the same shape (name · use ·
silhouette cues · typical materials).

---

## 가방 (Bag)

### Backpack — 신학기

- **Light String** — drawstring closure with padded straps, mesh side
  pockets, light synthetic body; positioned for high-school / early
  college daily commute
- **Round Zipper** — single round-curve zipper across the front face,
  printed wordmark or character placement, mid-weight; positioned for
  street-coded daily backpack
- **Round Pocket** — front round-shape pocket panel, optional flap,
  often paired with character keyring; positioned for whimsy / IP
  showcase

### Cross / sling

- **Soft sling** — single-strap, soft-body, conformable; daily
  loose-fit
- **Structured cross** — boxy cross with structured base; weekend
  out-of-home
- **Belt-bag / waist** — fanny / waist; festival / activity

### Tote / mini

- **Canvas tote** — heavyweight canvas, large logo or character
- **Mesh tote** — sporty light tote with mesh side
- **Mini mini** — palm-size charm bag

### Daily easy bag

- **Easy crossbody mini** — under-arm or short cross, light
- **Easy zip pouch** — flat zip pouch for small carry

## 모자 (Cap)

- **볼캡 / Ball cap** — 6-panel, structured or unstructured
  (washed / soft / camp variants)
- **버킷 / Bucket** — wide brim, soft body
- **메쉬 캡 / Mesh cap** — trucker-style with mesh back
- **비니 / Beanie** — knit, slouched or fitted
- **스냅백 / Snapback** — flat brim with snap closure
- **리본캡 / Ribbon cap** — soft-body ball cap with ribbon back-tie
  (women bridge style)
- **러플백 / Ruffle bag-hat** — soft brim with ruffle trim (women
  romantic line)

## 신발 (Shoes)

- **Sneaker (low) / 로우 키테** — low-cut sneaker, classic or modern
  silhouette; signature style
- **Sneaker (mid) / 미드 키테** — ankle-cut sneaker
- **Sandal — strap** — single or double strap on EVA / cushion sole
- **Sandal — slide** — open slide on cushion sole
- **Slide — one-mold** — one-piece molded slide (lightweight, summer
  hero)
- **Loafer / mule** — closed front, open heel
- **Boots** — chelsea / chukka / lace-up
- **Mary Jane** — strap closure, women bridge
- **Sock-shoe** — knit upper sock-style runner

- **Knee-high** — over-knee, women bridge

## 키링 / 참 (Charm / Keyring)

- **Character mascot** — brand mascot in plush / acrylic / silicone
  / jelly forms (Kiky / Lilly / Leo / Poco / Ace / Bear / Doroo /
  Puro / Pisisik in WACKYWILLY world)
- **Box pouch** — small zip pouch on key ring
- **Acrylic stand** — flat acrylic photo-card style
- **Sticker pack** — 3–5 sticker set with mascot

## Family card shape

```
Name              — short, internal (Korean + English when applicable)

Use               — 1-line "use when" / target customer
Silhouette cues   — 2–4 silhouette descriptors
Typical materials — 2–4 material options
SKU per family    — typical 2–5 in a season
Hero flag         — yes / no (max 1 per lineup)
```
